This Datapack is combined with a resource pack, under the name CustomMusic. Both the datapack and resource pack need to be used in tandem with eachother for
you to see the effect.

This Datapack and Resourcepack combination creates new custom disc's in the game which can all be obtained in many different ways depending on the disc you 
are looking for.

All discs can be obtained with commands using the built in functions that come with the Datapack, it will be a long list which have the following naming 
structure of custommusic:give_??????, where the string of "?" are what disc is being choosen to be given to the player.

The following will contain a list of all the disc's and how to obtain them within Survival minecraft:

Berdly:
    Can be obtained from using a smithing table and combining blue concrete with an observer and a gold ingot.

Big_Shot:
    Can be obtained from using a smithing table where you have purple concrete as the template the middle slot being the "Spamton" disc and the material slot being pink concrete.
